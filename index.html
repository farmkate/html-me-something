Starting Out

If you've never owned a farm before, you start off with the things that you think a farm should have.

  • You start off with chickens
  • Then you add some alpacas for good measure
After a few years

You learn a few things that would have been nice to know before you started, and some things that only experience can teach.

  • Roosters really can crow anytime during they day...or night
  • They really did use some alpaca (or llama) vocalizations for the dinosaurs of Jurassic Park
  • If you build it, the mice will come
New friends

And you find a few more friends along the way

  • Some goats become temporary residents
  • Some sweet little puppies come to stay forever
